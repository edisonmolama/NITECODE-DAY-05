1. git clone adalah cara menyaling repository

2. git init : adalah mengecek projek

3. git status : adalah mengecekprojek sudah terbaca atau tidak terbaca

4. git add . : adalah printa menyelek file dari folder

5. git commit -m " isi pesesan" : adalah memberikan nama file yang di push

6. git push : adalah exsport projek ke internet

7. git pull : adalah jika projek mengulang export ke internet

8. git clear : adalah hapus unutk mengunakan git bash

9. git cls : adalah hapus utntuk comment promt atau emd
